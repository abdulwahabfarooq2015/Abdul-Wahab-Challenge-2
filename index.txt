<b>There are 3 types of lists:</b><br /><br />
<dl>
<dt>definition lists</dt>
<dd>which show terms and definitions</dd>
<dt>unordered lists</dt>
<dd>which show bullets</dd>
<dt>ordered lists</dt>
<dd>which show numbers or letters.</dd>
</dl>
<p>
Above is an example of a definition list.</p>
<p>
Unordered lists can have a type attribute<br /> which can be applied to the UL or the LI tag.<br />
The three values allowed are:</p>
<ul>
<li type="square">type = "square"<br /></li>
<li type="disc"> type = "disc"<br /></li>
<li type="circle">type = "circle"<br /></li>
</ul>
<b>Ordered lists can have a type attribute too.<br />
Used along with a start attribute they can show:</b><br />
<ol>
<li type="a">lower case letters</li>
<li type="A">upper case letters</li>
<li type="i">lower case roman numerals</li>
<li type="I">upper case roman numerals</li>
<li>number</li>
</ol>
